# Product Management

Carpeta orientada a los roles y responsabilidades de la construcción y entendimiento del producto, desde la investigación de mercado hasta el descubrimiento constante.

## Contenido

- **continuous-discovery**: Hábitos para el descubrimiento continuo del producto.
- **inspired-product**: Cómo crear productos tecnológicos que los clientes amen (Marty Cagan - Inspired).
- **jobs-to-be-done**: Teoría para entender exactamente para qué te contrata un cliente.
- **mom-test**: Reglas y preguntas para validar ideas de producto sin recibir mentiras (The Mom Test).
