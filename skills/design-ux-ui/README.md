# Design / UX / UI

Esta carpeta reúne recursos, principios, libros y lineamientos enfocados en diseño visual, diseño de experiencia de usuario (UX) e interfaces gráficas.

## Contenido

- **design-everyday-things**: Resúmenes y aplicaciones de "La psicología de los objetos cotidianos" (Don Norman).
- **design-sprint**: Metodologías para ejecutar Design Sprints.
- **hooked-ux**: Prácticas para construir productos y experiencias que formen hábitos (Nir Eyal).
- **ios-hig-design**: Basado en los Human Interface Guidelines (Guías de Interfaz Humana) de iOS.
- **lean-ux**: Integración ágil de la experiencia de usuario y desarrollo.
- **microinteractions**: Diseño y valor de pequeñas animaciones e interacciones guiadas.
- **refactoring-ui**: Técnicas prácticas para mejorar el diseño sin ser diseñador.
- **top-design**: Prácticas globales de diseño.
- **ux-heuristics**: Leyes y heurísticas para validar interfaces (Jakob Nielsen y otros).
- **web-typography**: Reglas para legibilidad y el uso correcto de tipografías web.
- **web-design-guidelines**: Directrices actualizadas para el diseño web responsivo.
