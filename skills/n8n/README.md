# n8n Skills

Esta carpeta contiene habilidades enfocadas en flujos de trabajo, configuraciones de nodos, sintaxis y patrones para el uso de n8n.

## Contenido

- **n8n-code-javascript**: Buenas prácticas y ejemplos en JavaScript para n8n.
- **n8n-code-python**: Buenas prácticas y ejemplos en Python para n8n.
- **n8n-expression-syntax**: Uso de la sintaxis de expresiones y fórmulas de n8n.
- **n8n-mcp-tools-expert**: Configuración de herramientas expertas en MCP (Model Context Protocol).
- **n8n-node-configuration**: Guía experta para la configuración de diferentes nodos.
- **n8n-validation-expert**: Estrategias de validación de datos dentro de n8n.
- **n8n-workflow-patterns**: Patrones avanzados para orquestar y estructurar flujos.
